# QA / SDET Portfolio

안녕하세요.  
API 테스트 자동화와 성능 테스트를 중심으로 **재현 가능한 테스트 환경과 안정적인 테스트 실행 구조 설계**에 관심을 가지고 있는 QA 엔지니어입니다.

단순히 테스트를 작성하는 것보다 **테스트가 반복 가능하고 예측 가능하게 실행되는 환경을 만드는 것**에 초점을 두고 있습니다.

부트캠프 프로젝트를 통해 다음과 같은 경험을 쌓았습니다.

- pytest 기반 API 테스트 자동화
- JMeter 기반 성능 테스트 설계 및 실행
- Jenkins + Allure 기반 CI 테스트 파이프라인 구축
- 테스트 환경 의존성과 리소스 lifecycle 관리
- 반복 실행 가능한 테스트 구조 설계
